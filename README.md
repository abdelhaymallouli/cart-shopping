# Cart Shopping (Dessert Shop)

A dynamic and responsive frontend e-commerce application for a Dessert Shop, built with **Vanilla JavaScript**, **HTML5**, and **CSS3**. This project demonstrates efficient DOM manipulation and state management to provide a seamless shopping experience.

- **Dynamic Product Listing**: Displays a variety of delicious desserts with images and prices.
- **Interactive Shopping Cart**:
  - Add items to cart with a single click.
  - Real-time quantity adjustments (increase/decrease).
  - Remove items from the cart.
  - Automatic total price calculation.
- **Order Confirmation**: A polished modal popup confirming the order with a summary of purchased items.
- **Responsive Design**: Optimized for various screen sizes using pure CSS.
- **State Management**: efficient in-memory state handling for cart operations.
